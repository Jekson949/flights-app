# Flights App (Vite + React + TS + Redux + MUI)

## Стек
- Vite
- React + TypeScript
- Material UI (MUI) + @mui/icons-material
- Redux Toolkit (корзина, localStorage persist)
- React Router (/, /flights/:id, /cart)
- Axios (GET `getFlights`, `getFlightById`)

## Локальный запуск
```bash
npm install
npm run dev
